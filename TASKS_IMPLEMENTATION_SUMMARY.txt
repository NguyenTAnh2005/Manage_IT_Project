🔒 PERMISSION MODEL / MÔ HÌNH QUYỀN
================================================================================

CREATE (POST /projects/{id}/tasks):
  - Requires: User authenticated + PM role
  - Check: is_pm = await check_project_ownership(db, project_id, current_user.id)

READ - List (GET /projects/{id}/tasks):
  - Requires: User authenticated + Project member
  - Check: has_permission = await check_project_permission(db, project_id, current_user.id)

READ - Detail (GET /tasks/{id}):
  - Requires: User authenticated + (Project member OR Assigned to task)
  - Check: has_permission = await check_task_permission(db, current_user.id, task_id)

UPDATE (PUT /tasks/{id}):
  - Requires: User authenticated + PM role
  - Check: is_pm = await check_task_pm_permission(db, current_user.id, task_id)
  - Allowed fields: title, description, status, priority, assigned_to

DELETE (DELETE /tasks/{id}):
  - Requires: User authenticated + PM role
  - Check: is_pm = await check_task_pm_permission(db, current_user.id, task_id)

📊 API ENDPOINTS
================================================================================

Endpoint 1: GET /projects/{project_id}/tasks
  Purpose: Lấy danh sách công việc của dự án
  Auth: Required
  Permission: Project member
  Response: List[TaskResponse]
  Status: 200 OK
  Errors: 401, 403, 404, 500

Endpoint 2: POST /projects/{project_id}/tasks
  Purpose: Tạo công việc mới
  Auth: Required
  Permission: PM
  Body: TaskCreate
  Response: TaskResponse
  Status: 201 Created
  Errors: 401, 403, 404, 422, 500

Endpoint 3: GET /tasks/{task_id}
  Purpose: Lấy chi tiết công việc
  Auth: Required
  Permission: Project member OR Assignee
  Response: TaskResponse
  Status: 200 OK
  Errors: 401, 403, 404, 500

Endpoint 4: PUT /tasks/{task_id}
  Purpose: Sửa công việc
  Auth: Required
  Permission: PM
  Body: TaskUpdate
  Response: TaskResponse
  Status: 200 OK
  Errors: 401, 403, 404, 422, 500

Endpoint 5: DELETE /tasks/{task_id}
  Purpose: Xóa công việc
  Auth: Required
  Permission: PM
  Response: {"message": "Công việc đã được xóa"}
  Status: 200 OK
  Errors: 401, 403, 404, 500
